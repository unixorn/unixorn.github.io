Installing Docker on FreeNAS 11.2

Docker on FreeNAS 11 It isn’t as simple as just clicking on for a docker service in the UI. You’re going to have to set up a VM yourself. Create Rancher VM Create a VM in the new UI, select Docker Host. This will create a Rancher VM that already has docker installed inside so you don’t have to hassle with installing. Go into the classic webui, select the VM you just created, then devices, and edit Pick the RAW item, select edit, then set a password. This will be the rancher user’s password. Back in the new UI, go into VM, select the Rancher VM, hit the … and select serial to see the serial console. At the end of the boot messages, it will print something like docker-sys: 172.18.42.1 eth0: 10.9.8.7 lo: 127.0.0.1 before it presents a login prompt. The username is rancher, and the password is what you set in step 3. Setup Network on the Rancher VM I chose to set the static address for my Rancher VM on my DHCP server.
